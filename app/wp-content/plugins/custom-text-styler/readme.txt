A simple plugin to customize text styling.

== Description ==

The best plugin for edit your text!
You can select many settings from CSS, apply them to text and get a shortcode with the settings you choose.
= List of these settings: =
* font-size
* font-family
* font-style
* text-decoration
* opacity
* text-shadow
* font-weight
* color
* background-color

= For example: =
* font-size: 24;
* font-family: Arial, sans-serif;
* font-style: Normal;
* text-decoration: none;
* opacity: 0,5
* text-shadow: 1px - Horizontal offset, 1px - Vertical offset, 1px - Blur radius, black - Shadow color;
* font-weight: Bold;
* color: Black;
* background-color: White;

== Installation ==

= Activate =
Just download this plugin and activate it.
After activation plugin created the table wp_custom_text_styler_settings when saving all your shortcodes and styles css.
You don't need anything to work. Activate, that's all you need to get off to a good start!

= Deactivate =
When you deactivate this plugin. It will stop working and the table will be deleted from the database.
